FOR date, from csv to R:
R base function as.Date handles this type of conversion nicely. The trick is that you have to provide the origin, which in Excel is December 30, 1899.
as.Date(data, origin = "1899-12-30")
if the data is daily data, date is not stable, then use
cpi$DATE=as.Date(cpi$DATE,origin = "1899-12-30")
cpi$Date<-as.POSIXct(cpi$Date,tz = "UTC")
if data is monthly or annually or seasonal, then create the ts date by using
ts(data, frequency = 12, start = c(2014, 1))
